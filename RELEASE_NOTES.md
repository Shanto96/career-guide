# Release Notes

## Version 1.0.0 - Initial Cloud Release (🚀 Live)

**Release Date:** April 26, 2026
**Live Project:** [https://career-guide-app-993696842847.us-central1.run.app/](https://career-guide-app-993696842847.us-central1.run.app/)

We are thrilled to announce the initial production release of the **AI Career Guide Application**! This application leverages state-of-the-art Generative AI and Retrieval-Augmented Generation (RAG) architectures to provide intelligent, contextual career coaching based directly on your resume.

### 🎉 New Features

*   **AI Resume Analysis & Summarization:**
    *   Upload PDF resumes seamlessly via the web interface.
    *   Automatic text extraction using `PyPDF2`.
    *   Generation of a comprehensive, structured professional summary (Objective, Skills, Experience, Education, Achievements) utilizing Google's **Gemini 2.5 Flash** model.

*   **Interactive Resume Q&A (RAG Pipeline):**
    *   Ask specific questions about the uploaded resume.
    *   Implementation of a localized RAG architecture:
        *   Text chunking via LangChain's `CharacterTextSplitter`.
        *   Dense vector embeddings generated locally via `HuggingFaceEmbeddings`.
        *   Efficient similarity search utilizing a local `FAISS` vector database.
    *   Context-aware answering generated by Gemini 2.5 Flash for highly accurate, grounded responses.

*   **Cloud Infrastructure & Deployment:**
    *   Successfully containerized and deployed to **Google Cloud Run**.
    *   Configured for heavy machine-learning workloads with 4Gi of memory and increased Gunicorn worker timeouts to support initial model initialization.

### 🛠️ Technical Enhancements & Fixes
*   **Resolved Out-of-Memory (OOM) Errors:** Fixed startup crashes during deployment by increasing the Google Cloud Run default memory limit from 512Mi to 4Gi to support HuggingFace and PyTorch dependencies.
*   **Fixed Worker Timeouts:** Increased Gunicorn worker timeout to 120s to safely allow the initial downloading of local HuggingFace embedding weights on container cold starts.
*   **Documentation:** Updated the `README.md` with complete architectural diagrams (Mermaid), GenAI workflow details, and project links.
*   **Git Integration:** Fixed UTF encoding issues during automated commit generation by transitioning markdown files to standard UTF-8 encoding.

### 📦 Dependencies Added
*   `Flask` (Web framework)
*   `langchain`, `langchain-google-genai`, `langchain-huggingface` (AI Orchestration)
*   `faiss-cpu` (Vector Database)
*   `PyPDF2` (Document Processing)
*   `gunicorn` (Production WSGI server)
*   `google-genai` (Google LLM Integration)
